# Identification and Diagnosis of Misreporting in Surveys

This repository contains the data and analysis code for the manuscript.

## Contents
- Rasch model estimation
- Person response function (PRF) construction
- Manhattan distance + PAM clustering
- Hanning smoothing procedure

## Data
The dataset used in this study is publicly available in this repository, along with the analysis scripts needed to reproduce the results reported in the manuscript.
